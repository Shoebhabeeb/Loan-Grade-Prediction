What is Unsupervised Learning and difficulties involved in unsupervied Learning and name a few unsupervised algorithms ?

Solution¶

Unsupervised learning infers patterns from a dataset without reference to known outcomes. Unsupervised learning can be used to uncover previously unknown pattern in data, however, these patterns may be poor estimates as compared to what supervised learning can achieve.

Difficulties involved in unsupervised learning It is not possible to fit a linear regression model, since there is no associated response variable to predict. Unsupervised learning is usually performed as part of exploratory data analysis. Assessing results obtained from unsupervised learning can be difficult, since there is no common mechanism to perform validation on individual data sets.

Unsupervised Learning algorithms K-Means Clustering, Hierarchical clustering, Neural Networks, Gaussian Mixture, Hidden Markov Model.

Question 2 [5 Pts]¶

What is a PCA [Principal Component Analysis] and When to use PCA [Principal Component Analysis] ?

Solution¶

PCA is Principle Components Analysis. PCA refers to the process of computing principal components, and usage of these components for understanding the data. PCA is an unsupervised learning method as it involves only set of features say X1, X2, . . . , Xp, and no associated response Y.

PCA can also be used as a tool for visualizing data i.e. visualizing the observations and visualizing the variables. PCA provides a way to visualize high dimensional data providing the most important information.

For example, PCA is predominantly used as dimension reduction technique. Domains like facial recognition, image compression and computer vision. PCA can also be used in finding data patterns in high dimensional data of finance, data mining, bioinformatics etc.

Question 3 [5 Pts]¶

How does a PCA work [Principal Component Analysis] ?

Solution¶

Steps of working of PCA a. Calculate the covariance matrix X of data points. b. Calculate the eigen vectors and corresponding eigen values. c. Sort the eigen vectors according to their eigen values in decreasing order. d. Choose first k eigen vectors which will be the new k dimensions. e. Transform the original n dimensional data points into k dimensions.

Question 6 [10 Pts]¶

What is clustering? Explain how K-Means Clustering Algorithm works?

Solution¶

Clustering in data mining is an unsupervised learning procedure whose goal is to find homogenous subgroups among the observations. A cluster of data objects can be considered as one group. Advantage of clustering over classification is that it is adaptable to changes and helps group useful features that distinguish different groups.

Clustering analysis is used in applications such as market research, pattern recognition, image processing and data analysis. As a data mining function, clustering serves as a tool to gain insight into distribution of data to observe characteristics of each cluster.

Working of K-means clustering:-

Objective of K-means clustering is to group similar data points and discover underlying patterns. K is the number of clusters and must be fixed in advance i.e. the number of centroids in the dataset. i. Randomly assign each sample to cluster from 1..K, until all the data points are assigned to a cluster. ii. Find centroid for each cluster i.e. the average of all samples in the cluster. iii. Reassign each sample to the nearest centroid. iv. The algorithm converges to local minimum of

The K-means clustering algorithm halts creating and optimizing clusters when either a. The centroids have stabilized i.e. no change in centroids values because the clustering is successful, or, b. The defined number of iterations is achieved. K-means clustering can be achieved programmatically using pythons Scikit-learn library.

Question 7 [10 Pts]¶

What are the Advantages and disadvantages of Clustering ALgorithms discussed in our class (K-Means,Hierchal)?

Solution¶

K-Means Advantages & Disadvantages: Advantages- i. Easy to implement. ii. With large number of variables, K-Means may be computationally faster than hierarchical clustering, if k is small. iii. K-Means may produce tighter clusters than hierarchical clustering. iv. An instance can change cluster when the centroids are recomputed. Disadvantages i. Difficult to predict the number of clusters i.e. the k value. ii. Initial seeds have strong impact on results. iii. Order of the data has impact on the final results. iv. Rescaling the datasets by normalization or standardization completely changes the results.

Hierarchical clustering Advantages & Disadvantages: Advantages- i. Hierarchical clustering outputs a hierarchy, i.e. structure that is more informative than unstructured set of clusters returned by k-means. Hence, it is easy to decide the number of clusters by looking at the dendrogram. ii. Easy to implement.

Disadvantages- i. It is not possible to undo a step already executed, i.e. once the instances have been assigned to a cluster, it can not be moved to another cluster. ii. Time complexity: It is not suitable for huge datasets. iii. Very sensitive to outliers.

Question 8 [5 Pts]¶

Which Clustering Algorithm is better K-Means or hierarchical Clustering? Explain with a proper example which is better algorithm?

Solution¶

According to me hierarchical clustering is better compared to K-Means clustering. Hierarchical clustering is more flexible and has fewer assumptions about the distribution of dataset.

K-Means clustering requires a sense ahead-of-time about the desired number of clusters i.e. k-value. Also, K-Means gives unintuitive results if the data is not well-separated into clusters. Results may vary if the k-value selected is not well suited to the shape of the data, i.e. k-value picked is too high or too low. Different initial values for centroids may result in different cluster groupings.

In contrast, Hierarchical clustering has fewer assumptions about the distribution of data. Distance between each pair of data points such as Euclidian distance needs to be calculated. Hierarchical clustering joins nearby points into a cluster and then successively adds nearest points to the nearest group. The output of the algorithm is a dendrogram. Interpretation from the dendrograms requires special attention. The distance between two clusters is the maximum distance between any pair of samples, one in each cluster.

Input variables:

related with the last contact of the current campaign:¶

Output variable (desired target):

17 - y - has the client subscribed a term deposit? (binary: "yes","no")
